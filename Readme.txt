Steps to run the application:
	
o	Prerequisites: Install Python and Flask.
o	Open System Terminal on the root folder of the project.
o	Run Command: Python prediction.py
o	Once you see the below message:
		Running on http://127.0.0.1:5000/ (Press CTRL + C to quit)
 
o	Open browser and enter http://127.0.0.1:5000/ in the URL to run the application.
o	Select the Community Area, Date, Time and click on Predict key.
o	The system would recommend the crime type that is likely to happen for the selected inputs.
